# Spor Salonu Yönetim Sistemi

Bu proje, Web Programlama dersi kapsamında geliştirilen bir Spor Salonu Yönetim ve Randevu Sistemidir.

# Proje Hakkında
ASP.NET Core MVC mimarisi kullanılarak geliştirilen bu projede;
- Antrenör ve üye yönetimi
- Randevu alma ve onaylama sistemi
- Dinamik çalışma saatleri yönetimi
- Yapay zeka destekli egzersiz önerisi modülü bulunmaktadır.

# Teknolojiler
- ASP.NET Core MVC (.NET 8.0)
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Identity (Üyelik Sistemi)
